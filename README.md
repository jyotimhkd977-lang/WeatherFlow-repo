# WeatherFlow

WeatherFlow is a clean weather dashboard inspired by Samsung Weather, built with plain HTML, CSS, and JavaScript.

## Setup

1. Add your OpenWeatherMap key in `config.js`, or click the `API key` button in the app.
2. Start a local server in this folder:

```bash
python3 -m http.server 4173
```

3. Open `http://localhost:4173`.

## Features

- Large animated temperature display
- Search any city
- Current temperature, humidity, wind, and feels-like details
- 5-day forecast in a horizontal scroll layout
- Glassy, minimal styling with soft ambient motion
